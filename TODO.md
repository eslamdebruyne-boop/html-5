# مشروع فاتورة + مخزون + أرباح وخسائر ✅ تم

## المهام المنجزة ✅
- [x] **نظام الفاتورة** 🧾
  - زر "فاتورة" أخضر في الـ header
  - مودال ببحث ذكي عن المنتجات
  - اختيار العدد مع حد أقصى (المخزون)
  - عرض الإجمالي والربح المتوقع
  - زر "تسجيل البيع" يخصم من المخزون
  - آخر 5 مبيعات في أسفل المودال

- [x] **إدارة المخزون** 📦
  - تخزين في localStorage (ميترجعش بعد Refresh)
  - بيع→ ينقص تلقائياً
  - إظهار المتوفر/نفد في نتائج البحث
  - يدعم 3 أنواع للحقول: "المخزون", "المعروض", "المخزن"

- [x] **نظام الأرباح والخسائر** 📊
  - زر "التقارير" في الـ header
  - اختيار الشهر من dropdown
  - عرض: إجمالي المبيعات, إجمالي التكلفة, صافي الربح/الخسارة, عدد القطع
  - جدول تفصيلي بكل عملية: التاريخ, المنتج, الكمية, سعر البيع, التكلفة, الربح/الخسارة
  - يحسب الفرق بين "سعر البيع" و "سعر الشراء" (بيع أغلى = ربح أخضر، أوطى = خسارة أحمر)

- [x] **واجهة مستخدم** 🎨
  - توافق مع التصميم الداكن الحالي
  - مودالات متحركة (slide + fade)
  - رسائل Toast بعد كل عملية بيع
  - Responsive (شغال على الموبايل)
